# Cloud Economics

## Overview
My introduction to AWS cost management and financial planning — explaining methods for obtaining accurate pricing estimates, calculating multi-service architecture costs using AWS Pricing Calculator, and evaluating strategies to optimize expenses across AWS services.

---

## Services I Worked With

- **Amazon Elastic Compute Cloud (EC2)** — primary compute service analyzed for cost optimization strategies and pricing calculations
- **AWS Pricing Calculator** — tool used to estimate costs for multi-service AWS architectures and compare pricing scenarios
- **AWS Budgets** — service used to monitor and manage AWS spending

---

## What I Demonstrated

Through this SimuLearn, I demonstrated knowledge of:

- Methods and best practices for obtaining accurate AWS pricing estimates
- AWS Pricing Calculator functionality for building cost models across multiple services
- Cost calculation methodologies for complex, multi-service AWS architectures
- Cost optimization strategies including instance type selection, purchasing options, and resource right-sizing
- Financial planning and budget forecasting for AWS deployments
- Trade-offs between performance, availability, and cost when designing cloud solutions
