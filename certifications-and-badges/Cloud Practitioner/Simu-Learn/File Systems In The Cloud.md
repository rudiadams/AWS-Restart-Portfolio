# File Systems in the Cloud

## Overview
My exploration of cloud storage solutions — evaluating AWS storage options, analyzing the key features and benefits of Amazon EFS, applying EFS solutions to real-world business scenarios, and configuring centralized file system access across EC2 instances.

---

## Services I Worked With

- **Amazon Elastic File System (EFS)** — fully managed, scalable network file system providing shared storage across multiple EC2 instances with high availability and durability
- **Amazon Elastic Compute Cloud (EC2)** — compute resources configured to access and utilize centralized EFS storage

---

## What I Demonstrated

Through this SimuLearn, I demonstrated knowledge of:

- AWS storage options and their appropriate use cases including block, object, and file storage
- The key features and architectural benefits of Amazon EFS for shared file system requirements
- EFS performance characteristics, scalability, and availability across Availability Zones
- Application of EFS solutions to business scenarios requiring centralized, shared storage
- EFS endpoint configuration and mount target setup for multi-instance access
- Security and access control for shared file systems across multiple compute resources
- Cost considerations and performance optimization for EFS deployments
