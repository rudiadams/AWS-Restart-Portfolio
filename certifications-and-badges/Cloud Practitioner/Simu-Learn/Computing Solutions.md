# Computing Solutions

## Overview
My deep dive into Amazon EC2 capabilities and optimization strategies — comparing instance families and types, analyzing scaling approaches, configuring connection methods, and evaluating instance selection based on specific workload requirements.

---

## Services I Worked With

- **Amazon Elastic Compute Cloud (EC2)** — virtual server infrastructure with diverse instance families and types optimized for different workload requirements, scaling strategies, and connection configurations

---

## What I Demonstrated

Through this SimuLearn, I demonstrated knowledge of:

- EC2 instance families and types, and their suitability for different workload requirements
- Scaling concepts including vertical and horizontal scaling strategies
- Implementation approaches for scaling EC2 infrastructure
- Connection methods and configuration options for accessing EC2 instances
- Workload analysis and matching to appropriate instance types for cost and performance optimization
- Trade-offs between instance selection, performance, and resource requirements
